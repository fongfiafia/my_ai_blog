## 3. AI Chat Functionality

Cursor's AI chat functionality allows you to interact directly with an AI assistant to ask programming-related questions or seek help.

### How to Use:

1. Open the AI chat panel using shortcut keys:
   - Windows/Linux: `Ctrl+L`
   - macOS: `Cmd+L`
2. Type your questions or requests in the chat box.
3. The AI assistant will provide answers or suggestions.

### Example Conversation:

User: How do I read a CSV file in Python?

AI Assistant: In Python, you can use the built-in `csv` module to read CSV files. Here's a basic example:

```python
import csv

with open('file.csv', 'r') as file:
    csv_reader = csv.reader(file)
    for row in csv_reader:
        print(row)
```

This code opens a file named 'file.csv' and prints its contents line by line. You can also use `csv.DictReader` to read each line as a dictionary, where the CSV header is used as keys.

User: Can you give me an example using DictReader?

AI Assistant: Of course, here's an example using `csv.DictReader`:

```python
import csv

with open('file.csv', 'r') as file:
    csv_reader = csv.DictReader(file)
    for row in csv_reader:
        print(row)  # each row is now a dictionary
        # assuming CSV has 'name' and 'age' columns
        print(f"Name: {row['name']}, Age: {row['age']}")
```

The advantage of this method is that you can access data in each column directly by column name, making the code more readable and maintainable.
