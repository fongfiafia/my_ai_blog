# Cursor 搭配Deepseek大模型平民套餐组合

## 概览
欢迎来到 Cursor IDE 集成 Deepseek 大模型的使用指南。本教程将帮助你了解如何在 Cursor 中配置和使用 Deepseek 模型，提升你的编程效率。

## 简介
Cursor 是一款强大的 AI 辅助编程 IDE，我们提到过Cursor提供2周的会员体验，2周后会限制每个月的使用额度和请求速度。

除非你升级至会员，而付费策略至少每个月20美金，对于初学体验的小白来说有一点压力，这里我们介绍如何切换到白菜价的 deepseek-coder api。

deepseek-coder 是代码能力接近 GPT-4 的模型，关键是价格非常便宜，输入 1 元/ 1 百万 tokens（命中缓存 0.1元百万 tokens），输出 2 元/ 1 百万 tokens，同时支持文生图、图生图、文生视频能力。
