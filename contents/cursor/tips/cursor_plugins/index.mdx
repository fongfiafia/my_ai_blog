# Cursor 插件推荐

## 概述

Cursor如果你看了我这篇[cursor如何设置成中文](https://www.lookai.top/cn/cursor/basic/cursor_language)的文章，一定知道了cursor中有很多插件。

这篇文章我来推荐几个我正在使用的，也建议大家安装的插件，能够极高的提高大家的效率

## 打开插件平台

cursor编辑器可以安装非常多的插件，帮助我们提高效率。我们点击如下图这个按钮，就能打开插件市场

![](https://raw.githubusercontent.com/fongfiafia/img_bed/master/2024/12/07/1733554571619-7d314bd1-52c9-4aae-ba48-873634313ff6.png)

## Atom主题

有很多同学问题，你的cursor界面好好看，不是那种纯粹的黑色，而是带有一点点灰色，这样不是很刺眼。这是因为我安装了一个主题。

我推荐大家使用atom one dark这个主题，当然了还有其他主题，大家可以搜索theme，一个一个尝试，找到自己喜欢的。

![](https://raw.githubusercontent.com/fongfiafia/img_bed/master/2024/12/07/1733556419288-c815a28a-1696-4d3b-bbe9-b7accffd4ec0.png)


## Auto save

Cursor本身有自动保存功能，如果大家没有特别配置的话，默认都是有自动保存的，但是cursor在切换windows窗口后，往往就不能自动保存了。

这个问题如果出现在你开发小程序上，会经常出现，因此我特别推荐大家安装这个插件


![](https://raw.githubusercontent.com/fongfiafia/img_bed/master/2024/12/07/1733556778847-a6ba2aeb-a00d-4090-9e5e-a0e68939151d.png)


## 微信小程序插件

如果你是开发微信小程序，那么这个插件，能够提供提供语法高亮和语法补全，这样更加方便你去查看代码。


![](https://raw.githubusercontent.com/fongfiafia/img_bed/master/2024/12/07/1733556931662-ef5489ee-e847-458f-9c04-4156eab3915a.png)

## Markdown文件预览

如果你在cursor中写文章，特别是我写博客文章，文章格式是markdown，那么一定需要一个预览能力，否则你不知道你的文章长什么样子。


![](https://raw.githubusercontent.com/fongfiafia/img_bed/master/2024/12/07/1733557018477-cacf176c-0273-4dc0-85c9-a455188e0dda.png)

安装好以后，打开md文件在右上角有个按钮，点击后就可以预览

![](https://raw.githubusercontent.com/fongfiafia/img_bed/master/2024/12/07/1733557206970-acb67870-23d1-4688-ac9e-7d349a4b8436.png)
